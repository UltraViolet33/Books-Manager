# Books CRUD PHP

This is a sample web app in PHP.

## What can we do ?
    Display books
    Create a book
    Update a book
    delete a book

## Technologies used :
    HTML5/CSS3
    Boostrap
    PHP
    SQL

I used a MVC pattern that I made myself.
No need a framework like Laravel of Symfony, this is just to practice CRUD operations with PHP and oriented-object programming.
